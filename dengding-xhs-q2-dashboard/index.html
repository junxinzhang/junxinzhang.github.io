下阶段建议

继续测试“名人/家长共鸣 + 具体学校”的爆款结构，标题直接给出冲突感和对象名。
把上中国际、明星孩子、家庭择校这类高传播主题拆成系列，做同类标题和不同角度复投。
常规内容重点看封面点击率和收藏分享，不只看单条观看。
视频承担破圈任务，图文承担信息密度和长尾搜索任务，二者不要用同一套 KPI 判断。

数据口径

  • 所有指标来自用户提供的 Excel 原始数据，缺失数值按 0 处理。
  • 曝光、观看量、点赞、评论、收藏、涨粉、分享均为小红书笔记导出口径。
  • 互动率 = 点赞+评论+收藏+分享 / 观看量；保存传播率 = 收藏+分享 / 观看量。
  • 数据存在单条超级爆款，因此看板同时展示总量口径和非爆款参考口径。
